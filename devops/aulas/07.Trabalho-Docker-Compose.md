# Trabalho

## Resumo sobre Docker e Docker Compose

---

# Proposta

Elabore um **resumo escrito** sobre Docker e Docker Compose, com base nas aulas e em pesquisas complementares.

O objetivo é demonstrar que você compreende:

- O que é Docker e para que serve
- O que é Docker Compose e como ele se relaciona com o Docker
- Como essas ferramentas são usadas no dia a dia do desenvolvimento

---

# Tópicos Obrigatórios

O resumo deve abordar os seguintes pontos:

1. **O que é Docker?** Explique o conceito de container e como o Docker resolve o problema do "funciona na minha máquina"
2. **Imagem vs Container** Qual a diferença entre os dois conceitos?
3. **Dockerfile** O que é e para que serve?
4. **Docker Compose** O que é, quando usar e qual problema ele resolve?
5. **Exemplo prático** Descreva com suas palavras o que acontece quando executamos `docker compose up`

---

# Formato da Entrega

- **Mínimo**: 30 linhas de texto (excluindo código)
- **Formato**: Arquivo `README.md` em um repositório **público** no GitHub
- **Pode incluir**: Exemplos de código (Dockerfile, docker-compose.yml) para ilustrar as explicações
- **Linguagem**: Português, com termos técnicos em inglês quando necessário
